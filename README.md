## Project Name
Expense Tracker Pro

## Project Scope
This project is a Python-based Expense Tracker designed to help users manage daily expenses efficiently. It allows users to add, view, categorize, and delete expenses while maintaining records in a JSON file.

## Features
- Add Expenses
- View Expenses
- Delete Expenses
- Expense Categories
- Date & Time Tracking
- Total Spending Calculation
- JSON Data Storage

## Technologies Used
- Python
- JSON
- Datetime Module

## How to Run
1. Open the project folder in VS Code.
2. Open terminal.
3. Run:
   python expense_tracker.py
